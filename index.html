Teste code review no github

Principais atalhos do terminal github

Git clone + url – cria um clone do repositório git, ou seja, tudo que está no repositório desejado será copiada na sua maquina
Cd + nome da pasta de trabalho - Atalho para acessar pasta
Code . – Abre a pasta do repositório git no VScode
Ls – Verifica todos os arquivos do seu diretório
Git status – indica as alterações que foram adicionados e que ainda não foram feitos upload para o github
Git add . – Inclui todos os arquivos (prepara) alterados para fila de commit do github
Git add + nome do arquivo – Inclui apenas o arquivo especifico (prepara) alterado para fila de commit do github
Git commit -m “descreve o que está enviando “ – prepara para o push
Git checkout -b + nome de uma nova branch diferente do main – muda da branch main (principal) para uma pasta alternativa
git push origin + nome da pasta criada - envia o arquivo para o repositorio de origem no github
